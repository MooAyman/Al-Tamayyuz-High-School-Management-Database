# Al-Tamayyuz High School Management Database (SQL Project)

## 📌 Overview
This project is a relational database system for managing a school's data, including students, teachers, and subjects.

## 🛠️ Technologies Used
- MySQL
- SQL (DDL, DML, DQL)

## 📊 Database Structure
The database includes 3 main tables:
- Students
- Teachers
- Subjects

## 📌 Features
- Create and manage database schema
- Insert sample data (30 students, 10 teachers, 6 subjects)
- Perform queries (SELECT, ORDER BY)
- Update records
- Rename tables

## 🧠 Key SQL Concepts Used
- CREATE DATABASE & TABLE
- PRIMARY KEY & AUTO_INCREMENT
- ENUM data type
- INSERT INTO
- SELECT queries
- UPDATE statements
- ALTER TABLE
